## CoinPilot AI

CoinPilot AI 是一個全自動化的比特幣每日分析系統，旨在消除人工搜集資訊與撰寫報告的時間成本。

### 系統架構

```
採集層 (Eyes)     → 從 API 收集市場數據
        ↓
大腦層 (Brain)    → AI 分析生成文章
        ↓
展示層 (Face)     → Hugo 靜態網站發布
```

### 數據來源

- **CoinGecko** - 比特幣價格、交易量、市值
- **Alternative.me** - 恐慌貪婪指數
- **Google News** - 最新市場新聞
